```
Some shell subroutines to shorten a string, which represents a file path name.

Usage
    __spath_do()            ( : | <name> ) <str>
    __spath_get_cols()      ( : | <name> )

Environment variables
    SPATH_MARK              Prompt mark in shortened paths. Default is " ... "
    SPATH_KEEP              na
    SPATH_LENGTH            Value to calculate the maximal length of the path; to
                            begin the shortening. Default is 35
```
